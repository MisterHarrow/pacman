Welcome to the MisterHarrow Pacman Helm Repository

This repository contains Helm charts related to the Pacman project/demo, managed by MisterHarrow.

Usage

To add this Helm repository to your Helm client, run the following commands:

helm repo add misterharrow https://misterharrow.github.io/pacman/
helm repo update misterharrow

You can then install charts using, for example: helm install my-release misterharrow/<chart-name> --version <chart-version>

Available Charts

  • pacman-helm-chart (Version: 0.1.0) - A Helm chart for the Pac-Man application.

The source code and chart definitions can be found in the MisterHarrow/pacman GitHub repository.
